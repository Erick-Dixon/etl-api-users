# ETL API Users

Projeto simples de ETL em Python que consome uma API pública, transforma os dados e salva em JSON.

## Tecnologias
- Python
- Requests
- JSON

## O que o projeto faz
- Extrai dados de uma API
- Filtra nome, email e cidade
- Padroniza email (lowercase)
- Salva em arquivo JSON

## Como rodar
```bash
python main.py
